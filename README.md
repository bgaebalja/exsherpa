## 📋 프로젝트 설명
- B개발자팀 프로젝트 EX셀파
- 기출 시험지와 B셀파에서 제작된 시험지를 CBT 환경에서 풀이할 수 있는 서비스
- 전체 회원과 자신이 풀이한 시험지와 문제의 통계와 차트를 확인할 수 있는 서비스
  <br><br>

## 📅 프로젝트 기간
<b>2024. 11. 2 ~ 2024. 11. 11</b>
<br><br>
